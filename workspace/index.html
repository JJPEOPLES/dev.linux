About This Environment

This is a Linux development environment with LXDE desktop accessible directly in your web browser.

The environment is based on Ubuntu 22.04 and includes common development tools.

Launch Linux Desktop

How to Use

Simply click the "Launch Linux Desktop" button above to start using the Linux environment in your browser.

When prompted, enter the password: password

No additional software or plugins are required - everything runs in your web browser!

Available Tools

  • LXDE Desktop Environment
  • Firefox Web Browser
  • Git
  • Python 3 with pip
  • Node.js and npm
  • Basic development tools (gcc, make, etc.)
  • Text editors: nano, vim

Workspace

Your work is saved in the /home/devuser/workspace directory.

This directory is persistent and will retain your files between sessions.
