Welcome to dev.linux.containers

About This Environment

This is a Linux development environment with LXDE desktop accessible via RDP.

The environment is based on Ubuntu 22.04 and includes common development tools.

How to Connect

You can connect to this environment using any RDP client:

  1. Open your RDP client (like Microsoft Remote Desktop)
  2. Connect to dev.k2lang.org:3390
  3. Use the following credentials:
    • Username: devuser
    • Password: password

Available Tools

  • LXDE Desktop Environment
  • Firefox Web Browser
  • Git
  • Python 3 with pip
  • Node.js and npm
  • Basic development tools (gcc, make, etc.)
  • Text editors: nano, vim

Workspace

Your work is saved in the /home/devuser/workspace directory.

This directory is persistent and will retain your files between sessions.
